# multilingual-streaming-asr
Multilingual streaming ASR using NVIDIA Nemotron 3.5 ASR, with English/Greek/Cantonese evaluation, Greek/Cantonese fine-tuning, latency benchmarking, and GPU profiling on DGX Spark.

## Setup

Install dependencies only after confirming the CUDA/PyTorch/NeMo compatibility for the DGX Spark environment.
